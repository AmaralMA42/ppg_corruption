# Public Goods Game with Corruption

Monte Carlo simulation of an evolutionary game theory model
with cooperators (C), defectors (D), and corrupt agents (P).

## Model
- Spatial PGG on square lattice
- Random sequential update
- Fermi rule

## Run

python src/pgg_corupt_anima.py

## Output
- data/subpop.dat
- figures/egalitanima.gif
